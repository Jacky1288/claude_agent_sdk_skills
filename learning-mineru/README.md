# MinerU Learning Path

Welcome to your comprehensive guide to learning **MinerU** - a high-performance, open-source document parsing tool that transforms complex documents into LLM-ready formats.

## What is MinerU?

MinerU is a powerful document parsing tool that converts PDFs, images, Word documents, PowerPoint presentations, and Excel spreadsheets into structured Markdown and JSON formats. It's specifically optimized for:

- **RAG (Retrieval Augmented Generation) pipelines**
- **LLM training data preparation**
- **Academic research** (with exceptional formula recognition)
- **Document digitization** at scale (109 languages supported)

## Why This Learning Path?

This guide takes you from zero to productive with MinerU in a structured, progressive way:

1. **Level 1**: Understand what MinerU is and when to use it
2. **Level 2**: Get it installed and running your first document
3. **Level 3**: Master the core concepts and architecture
4. **Level 4**: Learn practical patterns for real-world use cases
5. **Level 5**: Explore advanced topics and next steps

## What You'll Learn

By the end of this learning path, you'll be able to:

- ✅ Install and configure MinerU on your system
- ✅ Choose the right backend (Pipeline, VLM, Hybrid) for your needs
- ✅ Parse PDFs, images, and Office documents into structured formats
- ✅ Extract tables, formulas, and multi-language text accurately
- ✅ Integrate MinerU into RAG pipelines and AI workflows
- ✅ Deploy MinerU via CLI, Python API, or REST API
- ✅ Optimize performance and troubleshoot common issues

## Prerequisites

- **Python 3.10-3.13** installed
- **Basic command-line knowledge**
- **16GB RAM minimum** (32GB+ recommended for VLM backend)
- **Optional but helpful**: GPU with 8GB+ VRAM for best performance
